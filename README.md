# Peniel Website

A static informational website built with HTML, CSS, and JavaScript. The project contains a series of pages covering religious and theological topics.

## Overview

This repository is a static site intended for educational or informational use. It includes a homepage plus pages for many subjects such as creation, faith and science, salvation, miracles, scripture, and more.

## Usage

1. Clone the repository.
2. Open `index.html` in a web browser.

Because this is a static site, no server or build step is required.
